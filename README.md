------------------ what the tool does -------------------------------

This is a tool that is used to scans for usernames on wordpress sites. 
it looks for usernames in various places like
    
    1. Rest API
     
    2. Rest route
     
    3. Author Archive
    
    4. Feeds
     
    5. sitemap XML
